GM University BCA Blogging Platform

This project implements a standards-compliant XHTML 1.0 Strict multi-page static website with core CSS and an external JavaScript file for dynamic behaviors.

Files added:
- `index.html` — Home page with dynamic welcome message
- `features.html` — Features list and a filterable table
- `about.html` — About/details and a small table
- `contact.html` — Contact form with real-time validation
- `css/style.css` — Single external stylesheet
- `js/scripts.js` — External JavaScript for dynamic features

How to use:
- Open the `index.html` (or any page) in a browser.
- The welcome message updates based on the current time.
- On `features.html`, type in the filter box to filter the features table.
- On `contact.html`, the form validates fields in real time and shows a confirmation alert on successful submission.

Notes:
- Files are written in XHTML 1.0 Strict markup with external CSS and JS.
- Footer displays the current year and the document last-modified date.
